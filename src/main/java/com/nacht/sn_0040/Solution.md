```azure
给定一个候选人编号的集合 candidates 和一个目标数 target ，找出 candidates 中所有可以使数字和为 target 的组合。

candidates 中的每个数字在每个组合中只能使用 一次 。

注意：解集不能包含重复的组合。 

 

示例 1:

输入: candidates = [10,1,2,7,6,1,5], target = 8,
输出:
[
[1,1,6],
[1,2,5],
[1,7],
[2,6]
]
示例 2:

输入: candidates = [2,5,2,1,2], target = 5,
输出:
[
[1,2,2],
[5]
]
 

提示:

1 <= candidates.length <= 100
1 <= candidates[i] <= 50
1 <= target <= 30
```

## 解题思路
同样的, 我们可以使用DFS+回溯来实现

首先, 我们把原来的数组排个序, 统计一下每个数各有多少个, 都丢到一个新的统计数组里面, e.g.
```azure
[
    (1, 1),
    (2, 2),
    (3, 3)
]
```
对应有1个1, 2个2, 3个3。

然后我们可以对这个新的数组做DFS+回溯
第一个位置, 我们可以选择0个1或者1个1, 分别做DFS
第二个位置, 我们可以选择0个2或者1个2, 或者2个2, 分别做DFS, 依次类推
