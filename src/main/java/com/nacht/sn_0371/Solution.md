```azure
给你两个整数 a 和 b ，不使用 运算符 + 和 - ​​​​​​​，计算并返回两整数之和。

 

示例 1：

输入：a = 1, b = 2
输出：3
示例 2：

输入：a = 2, b = 3
输出：5
 

提示：

-1000 <= a, b <= 1000
```
## 解题思路
```azure
我们以13 1101和11 1011为例子
首先明确两点:
1. 两个数做异或操作, 相当于全部位都做没有进位的加法, 比如例子中:
1101 ^ 1011 = 0110
2. 两个数做与操作, 然后左移1, 相当于获取所有进位, 比如例子中:
1101 & 1011 = 1001, 左移一位后: 10010
那么, 我们将无进位加法的结果和进位结果相加, 其实就是我们要的结果: 0110 + 10010 = 6 + 18 = 24 = 13 + 11
但是我们不是这样直接加起来, 只是说明它的关系。
实际上要不用加法求和, 就是下一步将我们的问题转成求0110和10010的和, 只需要继续分别计算出无进位加法结果和进位结果, 一直到进位结果等于0, 此时无进位加法结果, 也就是异或结果就是我们要的两数之和。
```
