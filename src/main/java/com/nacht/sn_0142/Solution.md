## 双指针问题

我们定义两个指针
- 一个指针我们称之为快指针, 该指针每次跳两步
- 一个指针我们称之为慢指针, 该指针每次跳一步

我们让快慢指针同时从头节点出发
- 如果快慢指针相遇了, 我们让快指针重新指向head, 并且改为每次走一步, 他们相遇的节点就是要找的进入环的节点
- 如果快指针走到了结尾, 两者没有相遇, 那说明链表没有环

## 如何求环的长度?
相遇时让快慢指针继续分别按2步/次, 1步/次的速度走, 第二次相遇时慢指针走的步数就是环的长度
